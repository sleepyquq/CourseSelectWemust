# WeMust 选课捡漏助手 🎓

这是一个基于 Selenium 开发的自动化选课脚本，专门为澳门科技大学学生解决**“想选的课没名额，需要随时守着网页等别人退选”**这一痛点。

## 🌟 核心价值：第一时间捡漏

在选课期间，热门课程往往瞬间满员。本脚本的作用不是“暴力抢课”，而是**“极速捡漏”**：
1. **无需时刻盯着电脑**：脚本会自动循环监控课程状态。
2. **秒级响应**：一旦有人退选放出名额，脚本会比肉眼发现和手动点击更快地执行“加选”。
3. **利用现有状态**：直接复用你 Edge 浏览器中已登录的 WeMust 账号，无需在脚本中输入密码，更安全、更方便。

---

## 🛠️ 使用场景

1. **错失先机**：选课开放初期没抢到心仪的课程。
2. **等待退选**：在选课结束前，希望守候可能出现的零星名额（有人退课或系统扩容）。
3. **已登录状态**：你已经在 Edge 浏览器上成功登录了 WeMust 账号。
4. **全天候运行**：脚本通过不断循环“进入页面 -> 寻找课程 -> 点击加选”，直到成功为止。

---

## 🚀 准备工作

### 1. 环境依赖
- **操作系统**：Windows (已内置支持 Edge)
- **Python 环境**：安装 Python 3.x
- **浏览器**：Microsoft Edge
- **驱动程序**：项目已包含 `msedgedriver.exe`（需确保与你的 Edge 版本匹配）

### 2. 安装必要库
在终端执行以下命令：
```bash
pip install selenium
```

---

## ⚙️ 配置与使用

### 第一步：关闭所有 Edge 窗口
由于脚本需要接管你的 Edge 用户数据目录（以便维持登录状态），运行前**必须关闭所有已打开的 Edge 浏览器窗口**。

### 第二步：修改脚本配置
打开 `auto_select_edge.py`，根据你的实际情况修改以下代码：

1. **用户路径配置**（第 20 行）：
   将路径中的 `ovo` 更改为你 Windows 系统的实际用户名。
   ```python
   options.add_argument("--user-data-dir=C:\\Users\\你的用户名\\AppData\\Local\\Microsoft\\Edge\\User Data")
   ```

2. **目标课程定位**（第 54 行 & 第 68 行）：
   - **修改分支**：将 `人文藝術` 改为你目标课程所属的分支名称。
   - **修改按钮索引**：`all_add_btns[2]` 代表点击该分支下的**第 3 个**“加选”按钮（索引从 0 开始）。
   ```python
   # 定位分支
   branch_btn = driver.find_element(By.XPATH, "//span[contains(text(), '目标分支名称')]")
   # 选择课程（例如第1个加选按钮则改为0）
   select_btn = all_add_btns[0] 
   ```

### 第三步：运行脚本
```bash
python auto_select_edge.py
```

---

## 🔄 运行逻辑说明

1. **自动启动**：脚本启动带插件和登录信息的 Edge 浏览器。
2. **跳转页面**：直接进入 WeMust 选课系统的“加选”界面。
3. **循环监控**：
   - 寻找指定的选课分支并点击。
   - 扫描页面上所有的“加选”按钮。
   - 尝试点击你预设的目标按钮。
   - **刷新重试**：如果未成功或报错，脚本会等待 10-15 秒后自动刷新页面，开启下一轮尝试。
4. **成功反馈**：一旦点击成功，脚本会停留在结果页面或报错异常处，建议用户及时人工确认。

---

## ⚠️ 注意事项

- **不保证 100% 成功**：脚本大大提升了捡漏概率，但如果网络波动或多人同时抢一个名额，仍存在失败可能。
- **防止封号**：脚本已预设了合理的等待时间（10-30秒不等）。**请勿过度缩短时间间隔**，以免触发系统反爬机制导致账号被封禁。
- **驱动匹配**：如果运行报错提示 `SessionNotCreatedException`，请前往 [Microsoft Edge Driver 官网](https://developer.microsoft.com/en-us/microsoft-edge/tools/driver/) 下载与你浏览器版本一致的驱动替换 `msedgedriver.exe`。
